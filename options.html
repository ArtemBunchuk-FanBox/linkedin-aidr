LinkedIn AI:DR

AI; Didn't Read — hides LinkedIn feed posts that mention AI or look AI-generated.

General
Topic / keyword filter

Each topic has its own name, keyword list, and on/off toggle. Posts that match any keyword from an enabled topic get flagged. Short tokens like ai match whole-word only, so "said" and "again" don't trigger.

Add a topic

Leave keywords empty to create the topic with just its name as a single keyword.

Heuristic "slop" detector

Counts signals like em-dash overuse, "it's not just X, it's Y", emoji bullets, hook openers, AI-favored phrases ("delve", "tapestry", "navigate the complexities"). Defaults to 3.

AI-detector API (optional)

Calls your chosen provider per post. Use borderline only mode to limit cost — posts already caught by keywords/heuristics skip the API entirely.

GPTZero is a dedicated AI-text classifier. See gptzero.me for API access.

API keys are stored locally in chrome.storage.local and sent only to the selected provider's endpoint. They never touch the LinkedIn page.
